PART 2: RUN THE GOLDEN SET MERGE CODE AS SHOWN#

  1. You should have both the AnnotationData and the DataProcessing repositories cloned to your local PC at the same level, and up to date (pull first).

  2. The project configuration for your files should be: –myGithubs —-AnnotationData —-DataProcessing

  3. Open up a GitBash window in your local DataProcessing repository. (Assumes you have cloned this repository to your local machine, and have set up GitBash. If you have not, please do so.) Right click on the DataProcessing folder, select ‘GitBash Here’

Note

IF YOU HAVE ANY PROBLEMS AT ALL RUNNING THE CODE, CONTACT JENNIFER IMMEDIATELY BEFORE YOU ATTEMPT TO ALTER THE CODE.

PART 3: CREATE and ASSIGN A NEW GOLDEN SET CSE PROJECT#

  1. Go to CSE, log in, and open the current NALT_ONLY_PROJECT_TEMPLATE_2022.

  1. Go to the pull down menu on the upper right, and select ‘Project backup…’. Select the recent November 13 update to NALT, and click the restore (arrow curving to the right) icon, circled below.

fig. 16

  1. Enter the information in the proper format for the new CSE golden set project, as shown. The project will take several minutes to be generated.

fig. 17

  1. Monitor progress, and wait for the creation process to complete. It may take up to 15 minutes +/-.

fig. 18 fig. 19

  1. Once completed, go to the ‘Home’ tab, and OPEN the new project. YOU MUST DO THIS STEP SO YOU ARE NOT UPLOADING DOCUMENTS TO THE TEMPLATE PROJECT. Double check that you have opened the new project before proceeding.

  1. Go to the ‘Documents’ tab, and click ‘ADD DOCUMENTS’.

fig. 22

  1. Select ‘Choose File’.

fig. 23

  1. Select the zip file you just created, and then click ‘Open’ on the file manager text box.

fig. 24

  1. Double check the filename and format, and click ‘ADD’, the files should load quickly.

fig. 25 fig. 26

  1. Once it is complete, go to the ‘Documents’ tab, and check that the documents loaded correctly, the correct number of documents, and that they are displaying properly.

  2. Assign the new golden set CSE project to the Golden_Review_Logon account. Go to the ‘Administration’ tab, select the Golden_Review_Logon account, and then select the ‘+’ in the right side of the page.

fig. 27

  1. Select the pull-down menu to find the new project and select it. Make the ‘Role’ and Editor and hit the ‘OK’ button to the right of the ‘Editor’ box. GO TO THE CENTER OF THE GUI, and HIT THE “SAVE” button! This assignment will NOT be saved if you do not select the ‘Save’ button once you have made these selections.
